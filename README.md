# 📦 Food Expiry Tracking Web Application

🌟 **Overview**  
This web application is designed to help users manage and track food items based on their manufacturing and expiry dates. It is especially useful for households and small shop owners to reduce food waste and efficiently manage stock.

---

## 🚀 Key Features
- 📝 Add and manage food items with manufacturing and expiry dates
- 📊 Automatic expiry status detection
- 🎨 Color-based notifications for easy tracking:
  - 🔴 Red – Items expiring within 2 days
  - 🟠 Orange – Items nearing expiry
  - 🟢 Green – Items with longer shelf life (more than 1 month)
- 🔢 Quantity management for each item
- ⚡ Real-time updates for inventory tracking

---

## 🛠️ Technologies Used
- **Frontend:** JavaScript, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Architecture:** MERN Stack

---

## 🎯 Purpose
This system helps users:
- Reduce food wastage
- Manage kitchen or shop inventory efficiently
- Easily identify items that need urgent attention
